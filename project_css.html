Netflix Clone

A full-stack Netflix clone with video browsing, authentication, and TMDB integration.

Project Overview

This is a Netflix-inspired full-stack project built using React, Tailwind CSS, Node.js, Express, and MongoDB. It includes user login, real-time movie data from TMDB API, and a streaming UI similar to Netflix.

Demo Video

Project Details

  • Frontend: React.js, Tailwind CSS
  • Backend: Node.js, Express.js, MongoDB
  • Authentication: Firebase
  • API: TMDB API for movie data
  • Features: Login, logout, movie browsing, and responsive UI
